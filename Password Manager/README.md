# Password Manager (C++)

A simple Password Manager built in C++ that allows users to securely store, view, and manage saved credentials.  
This project demonstrates file handling, user input validation, and modular program design.

---

## Features
- Add new passwords (website, username, password)
- View stored passwords
- Master password verification
- Local file storage (`passwords.txt`)
- Simple and user-friendly text interface

---

## Technologies Used
- **C++**
- File I/O (read/write)
- Basic string manipulation
- Modular programming (header and source files)

---

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/ikonbegins/password-manager-cpp.git
2. Open the project in VS Code or your preferred IDE.
3. Compile the program:
    ```bash
    g++ main.cpp password_manager.cpp -o password_manager
4. Run the executable:
    ```bash
    ./password_manager
